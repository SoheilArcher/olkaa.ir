# Lessons Learned

Version: v1.0.0  
Status: Initial structure

This folder stores lessons learned from Fava Imen Olka projects, incidents, deployments, integrations, and operational work.

## Naming

Use sequential lesson records:

- `LESSON-0001.md`

## Template

```markdown
# LESSON-0001: Lesson Title

## Problem

Describe the problem, incident, mistake, delay, or unexpected outcome.

## Root Cause

Explain why it happened.

## Solution

Describe how it was resolved.

## Prevention

Explain how the company should prevent the issue from happening again.

## Related Products

List related products, systems, clients, or teams.
```

## Rules

- Write lessons professionally and without blame.
- Include practical prevention steps.
- Never delete historical lessons.
